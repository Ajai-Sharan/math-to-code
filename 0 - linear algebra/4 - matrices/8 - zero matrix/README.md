# Zero Matrix

## Pure Python

### version 1 --> A. List Multiplication Trick ([[0]*cols for _ in rows])

### version 2 --> B. Nested List Comprehension (Explicit Loop)

## NumPy (The Industry Standard)

### version 3 --> A. numpy.zeros() (Standard Initialization)

### version 4 --> B. numpy.zeros_like() (Copy Shape from Existing)

This is very useful in Machine Learning. It creates a zero matrix with the same shape as another existing matrix.

## Deep Learning Libraries

### version 5 --> A. PyTorch (torch.zeros or zeros_like)

### version 6 --> B. TensorFlow (tf.zeros or zeros_like)
